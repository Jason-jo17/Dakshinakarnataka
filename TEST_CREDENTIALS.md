## How to Use These Credentials

### For Institutions:
1. Go to **Credential Manager** → **Generate Credentials** tab
2. Select **Institution** as Partner Type
3. Select the training center from the dropdown
4. Click **Generate Credentials**
5. Use the generated username/password to log in

### For Trainees:
1. Go to **Credential Manager** → **Generate Credentials** tab
2. Select **Trainee** as Partner Type
3. Enter trainee name (e.g., "Rajesh Kumar")
4. Select the training center to link them to
5. Click **Generate Credentials**
6. Use the generated username/password to log in

### Login Process:
1. Go to the main login page
2. Select the appropriate tab (Institution or Student)
3. Enter username and password
4. Click Login
5. You'll be redirected to your dashboard with pre-filled data
